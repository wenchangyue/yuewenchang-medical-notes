# 岳文昌医生｜中文医学科普资料库

岳文昌医生 YouTube 频道的公开 companion site，用 GitHub Pages 托管经核对的医学文字、适用边界、复核日期和来源。

- 站点：<https://wenchangyue.github.io/yuewenchang-medical-notes/>
- YouTube：<https://www.youtube.com/@yuewenchang>
- Channel ID：`UChgLMLpKOV24mFEhGbp1diQ`

## 发布一页视频资料

1. 视频必须已经公开，并找到最终成片对应的校对字幕。
2. 复制 `TEMPLATE-video.md` 到 `_videos/<slug>.md`。
3. 核对药名、试验名、疾病分期、样本量、数字和来源。
4. 明确页面是逐字稿还是经核对的文字整理。
5. 检查 JSON-LD、canonical、链接、`robots.txt`、`sitemap.xml` 和 `llms.txt`。
6. GitHub Pages 构建成功后，把页面链接加入对应 YouTube 简介。

未公开视频不建立可索引页面。有待核数字或来源缺口的视频，等完成核验后再收录。
